# Generic styles
```
    .container -> max 1200px and and puts it to middle
    .row -> flex box
    .column -> flex item
    .center -> flex center
    .italic -> text to italic
    .light -> text to light
    .normal, .regular -> text to normal
    .bold, .heavy -> text to heavy (500)
    .block  -> display block
    .inline-block -> display inline block
    .no-outline -> no-outline on all focus, hover etc
    .relative -> position relative
    .op75  -> opacity 0.75
    .op100 -> opacity 1
    .transparent -> background color transparent
    .over-hidden -> overflow hidden
    .capitalize  -> capitalize text
```
